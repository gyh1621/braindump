## Solution {#solution}

```java
class Solution {
    public boolean canJump(int[] nums) {
        int maxDis  = 0;
        for (int i = 0; i < nums.length && i <= maxDis; i++) {
            maxDis = Math.max(i + nums[i], maxDis);
            if (maxDis >= nums.length - 1) return true;
        }
        return false;
    }
}
```


## Complexity {#complexity}

-   time: O(N)
-   space: O(1)
