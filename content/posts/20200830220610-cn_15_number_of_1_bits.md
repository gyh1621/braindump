## Solution 1 - Loop {#solution-1-loop}

```java
public class Solution {
    // you need to treat n as an unsigned value
    public int hammingWeight(int n) {
        int sum = 0;
        while (n != 0) {
            if ((n & 1) == 1) sum++;
            n >>>= 1;
        }
        return sum;
    }
}
```


### Complexity {#complexity}

-   time: O(log2N)
-   space: O(1)

```java
public class Solution {
    // you need to treat n as an unsigned value
    public int hammingWeight(int n) {
        int sum = 0;
        while (n != 0) {
            sum++;
            n &= (n - 1);
        }
        return sum;
    }
}
```
